# QuraMate

A modern, open-source database management tool. Connect. Query. Manage.

## Install

```bash
# npm
npm install -g quramate

# pnpm
pnpm add -g quramate

# yarn
yarn global add quramate

# bun
bun add -g quramate
```

## Launch

```bash
quramate
```

## Supported Platforms

| Platform | Architecture |
| -------- | ------------ |
| Windows  | x64          |
| macOS    | Universal    |

## Supported Databases

PostgreSQL, MySQL, MariaDB, SQL Server, SQLite, LibSQL, CockroachDB, Greenplum, Amazon Redshift, Databend
