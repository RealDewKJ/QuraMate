# QuraMate Project Capabilities

QuraMate is a modern, cross-platform database client built with Go (Wails) and Vue.js, designed for speed and ease of use. Key features include:

## 1. Supported Databases

Connect to major relational databases:

- **PostgreSQL**
- **MySQL**
- **SQL Server (MSSQL)**
- **SQLite**

## 2. Connection Management

- **Saved Connections**: Automatically remembers connection details for quick access.
- **Test Connection**: Validate server status and credentials before connecting.
- **UI Themes**: Supports both Light and Dark modes.

## 3. Dashboard & Data Explorer

- **Table Navigation**: Browse all database tables via the sidebar with a search filter.
- **Multiple Tabs**: Open multiple query or data tabs simultaneously.
- **Context Menu**: Right-click on tables for quick actions:
  - **Select Top 100**: Instantly view the first 100 rows.
  - **View Design**: Inspect table schema and column definitions.
  - **View ER Diagram**: Visualize table relationships.

## 4. Data Grid

- **Virtual Scrolling**: Smoothly handle and display large datasets.
- **Inline Editing**: Edit cell data directly by double-clicking and pressing Enter to save.
- **Sorting & Filtering**: Sort data by column and apply filters to find specific records.

## 5. SQL Tools

- **Advanced SQL Editor**: Powered by Monaco Editor (VS Code), featuring syntax highlighting and autocomplete.
- **Query Beautifier**: Format SQL code for better readability with a single click.
- **Performance Stats**: Track query execution time.

## 6. Visualization

- **ER Diagram**: Generate Entity-Relationship Diagrams from foreign key constraints to visualize database structure.
