# Third-Party Licenses

QuraMate is built on top of open-source software.  
Please review upstream licenses before redistribution.

## Frontend Dependencies (NPM)

- Dependency lock file: `frontend/package-lock.json`
- Main package manifest: `frontend/package.json`

Primary libraries include:

- Vue
- Vue I18n
- VueUse
- Radix Vue
- Monaco Editor
- Mermaid
- sql-formatter
- Tailwind CSS

## Backend Dependencies (Go)

- Dependency manifest: `go.mod`
- Checksums and resolved modules: `go.sum`

Primary libraries include:

- Wails
- pgx
- go-sql-driver/mysql
- go-mssqldb
- modernc.org/sqlite
- excelize
- go-keyring
